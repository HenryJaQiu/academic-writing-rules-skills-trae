# 项目覆盖规则模板

本文件只保留 **当前项目特有** 的规则，用来覆盖通用规则与 venue 规则。

分层说明：

- 通用层：`.trae/rules/academic_base.md`
- venue 层：`.trae/rules/venue_rules.md` 或具体 venue 文件
- 项目层：本文件

## 1. 工作流

- 默认只更新必要产物，不自动做全文编译
- 只有当用户明确要求编译，或需要验证图放入论文后的真实显示效果时，才执行全文编译
- 每轮修改优先只动一个高杠杆问题，避免多方向混改

## 2. 图表与脚本映射

- 当用户要求修改特定图时，必须先建立“图号 -> 图文件 -> 生成脚本”的映射表
- 未确认图号前，不允许跨脚本改图
- 任何图内字体、legend、边界、长宽比调整后，都要检查导出的图 PDF；若影响正文真实显示效果，再编译最终论文 PDF 复核
